Add subscribers to a user-provided MailChimp list after checkout with Paypal

== Description ==

MailChimp for Paypal Shopping Cart addon allows you to add subscribers to your MailChimp newsletter list after a customer purchase items from your site. 

Options are provided to use Double Opt-In confirmation, update existing subscribers, update or replace subscribers' interest groups, and automate sending of your list's welcome message.

If you are using [PayPal Shopping Cart](https://www.tipsandtricks-hq.com/wordpress-simple-paypal-shopping-cart-plugin-768) plugin and you would like to integrate MailChimp with it you can use this addon plugin.

You can find detailed usage instructions for this addon [here](https://www.tipsandtricks-hq.com/ecommerce/wp-shopping-cart-and-mailchimp-integration-3442)

== Installation ==

1. Upload 'mailchimp-for-paypal-shopping-cart' folder to the '/wp-content/plugins/' directory [OR] Upload 'mailchimp-for-paypal-shopping-cart.zip' plugin via  the 'Plugins' menu in WordPress (Dashboard > Plugins > Add New > Upload Plugin).
2. Activate the plugin via the 'Plugins' menu in WordPress.
3. Plugin settings are now located in 'Settings', 'MailChimp Settings'.

== Frequently Asked Questions ==

= How do I provide mailing list data? =

First, you need to create a free Mailchimp account. Once you have an account, you can create a list to which this addon can add your customers.

= How do I set up the plugin? =

1. Install and activate the plugin.
2. Go to the plugin settings page (Dashboard > Settings > MailChimp Settings) and edit the necessary options.

= How do I find my List ID? =

In your MailChimp account, open your list (or create a new list, if you don't have one yet) and select 'List Name & Defaults' from the 'Settings' menu. You should see the List ID displayed.

= How do I find my API Key? =

1. In your MailChimp account, open your account/user page and select 'API Keys' from the 'Extras' menu.
2. Scroll down to 'Your API Keys' and click 'Create a Key'.

For more information please visit the [addon documentation page](https://www.tipsandtricks-hq.com/ecommerce/wp-shopping-cart-and-mailchimp-integration-3442)
